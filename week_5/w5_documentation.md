# Week 5 Tutorial 5 - Cafe Bill Calculator Documentation

## 1. Problem Analysis

### 1.1 Problem Statement

The problem is to create a Python program that helps a small cafe calculate customer bills automatically. 
The program calculates the total price based on the quantity of coffee, tea, and sandwich ordered by the customer.

---

### 1.2 Inputs

The program requires the following inputs:

- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

---

### 1.3 Outputs

The program will display:

- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity
- Total bill amount

---

### 1.4 Typical Process Flow

The typical process flow is:

1. Start the program.
2. Get customer information.
3. Ask the customer for item quantities.
4. Calculate the cost of each item.
5. Calculate the total bill.
6. Display the receipt.
7. End the program.

---

### 1.5 Constraints

The program has the following constraints:

- Only available items are coffee, tea, and sandwich.
- Prices are fixed.
- Quantity must be a valid number.
- The program only calculates one customer's bill at a time.

---

# 2. Problem Decomposition

The problem can be divided into smaller tasks:

1. Create item prices.
2. Get customer input.
3. Calculate the cost for each item.
4. Calculate the total bill.
5. Print the customer receipt.
6. Test the program output.

---

# 3. Pseudocode

START

Input customer name

Input tea quantity

Calculate coffee cost:
coffee quantity × coffee price

tea quantity × tea price

Calculate sandwich cost:
